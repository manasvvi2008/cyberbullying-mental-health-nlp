# Cyberbullying & Mental Health Detection using NLP and XAI

## Problem Statement
The exponential rise in social media usage has escalated both mental health 
crises and cyberbullying incidents globally. While these issues are deeply 
interconnected, existing studies address them in isolation. This study 
proposes a unified machine learning framework using NLP and Explainable AI 
to simultaneously detect cyberbullying and classify associated mental health 
conditions such as depression, anxiety, and stress from social media text 
data, enabling early and transparent intervention.

## Datasets
1. Cyberbullying Classification Dataset (Wang et al. / Andrew MVD)
2. Sentiment Analysis for Mental Health (Sarkar, 2024)
3. Suicide and Depression Detection (Komati)
4. Reddit Mental Health Data (Ghoshal, 2023)

## Planned Models / Methods
- BERT, BERT+CNN, BERT+BiLSTM (hybrid architectures)
- Multi-task learning framework (joint cyberbullying + mental health classification)
- Explainable AI: SHAP, LIME, Attention Visualization (TIBAV)
